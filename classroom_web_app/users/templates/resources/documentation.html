ClassConnect Documentation

ClassConnect is a DBMS-driven virtual classroom tool designed for seamless communication between students and instructors.

🔐 Authentication

  • Secure registration with email verification
  • Password recovery via email
  • Two-step verification on login

📚 Classroom Management

  • Create, manage, and delete classrooms
  • Handle student enrollment and invitations
  • Announce tasks and view analytics

📝 Assignments & Quizzes

  • Create assignments with rich descriptions and files
  • Schedule quizzes, auto-publish, grade, and analyze
  • CSV export and leaderboard view

💬 Discussion & Notifications

  • Topic threads with replies and reactions
  • Instructor pinning and priority tags
  • Notifications on assignment updates
